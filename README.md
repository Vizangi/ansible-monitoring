# Ansible Monitoring

## Добро пожаловать в репозиторий Ansible

### Начало
- Глобально мы поднимем бэкенд мониторинга на основе **Prometheus** с помощью автоматизации **Ansible**
- И всё это разрезе **Linux** (Terminal Debian)
